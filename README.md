# Minimal Foo portfolio site

This is a plain HTML/CSS website. No build step, no packages, no JavaScript.

## Files

- `index.html` — landing page with the two choices
- `laugh.html` — comedy / performance page
- `learn.html` — talks, workshops, and Udemy page
- `styles.css` — all styling
- `assets/` — images pulled from the mockups

## Edit these before publishing

Open the HTML files and search for these placeholders:

- `Foo` — replace with your real name if needed
- `https://www.instagram.com/foo_comedy/` — replace with your actual comedy dates link
- `https://example.com/meat-cute-or-die-tickets` — replace with the real ticket link
- `https://example.substack.com` — replace with your Substack
- `https://www.udemy.com/` — replace with your actual Udemy course link
- `your.email@example.com` — replace with your real email address

## Host on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files and folders to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/root` folder.
6. Save.

GitHub will give you a public URL after the site deploys.
